# webcam2avatar

Browser-based face detection and personalized SVG avatar generator. 100% client-side, open source.

## [Live Demo](https://claudesoft.github.io/webcam2avatar/)

## How it works

1. Take a webcam photo
2. Face-api.js detects age, gender, expression, skin tone & hair color
3. Three unique SVG avatars are generated locally based on your features

Everything runs in the browser — no images or data leave your machine.
